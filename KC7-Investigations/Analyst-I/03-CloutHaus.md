# Investigation 03 — CloutHaus
**Badge:** Clout Defender
**Badge ID:** 64451
**Date:** June 05, 2026
**Platform:** KC7 Cyber
**Path:** Security Analyst I — Module 3

---

## Scenario Summary
Investigated a targeted phishing and social
engineering attack against a rising influencer.
Uncovered how personal details shared on
social media were exploited to compromise
her accounts.

---

## Techniques Used
- OSINT — gathering public information
- Phishing analysis
- Log forensics
- KQL query writing
- Passive DNS analysis
- Traffic analysis (inbound/outbound)

---

## What I Investigated
- Employee logs
- Inbound and outbound network traffic
- Passive DNS data
- Social media oversharing patterns
- Attacker infrastructure

---

## Key Findings
- Personal social media details enabled attack
- Attacker used phishing as entry point
- DNS data revealed attacker infrastructure
- Traffic analysis confirmed compromise

---

## MITRE ATT&CK Techniques
| Technique | ID | Description |
|---|---|---|
| Phishing | T1566 | Initial access via phishing |
| OSINT | T1593 | Search open sources |
| DNS | T1071 | Attacker C2 via DNS |

---

## What I Learned
- OSINT is powerful for both attackers
  and defenders
- Social media oversharing creates real
  security vulnerabilities
- DNS logs reveal attacker infrastructure
- Digital forensics connects all evidence

---

## Real World Connection
> Social engineering attacks targeting
> individuals with public profiles are
> increasingly common. This investigation
> mirrors real cases where influencers,
> executives and public figures are targeted
> using their own publicly shared information.
> OSINT skills are essential for any
> SOC analyst investigating such cases.
