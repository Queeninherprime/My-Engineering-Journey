---

## 1. OSI Model
| Layer | Name | Protocols |
|---|---|---|
| 7 | Application | HTTP, DNS, SMTP, FTP |
| 6 | Presentation | SSL/TLS, encryption |
| 5 | Session | NetBIOS, RPC |
| 4 | Transport | TCP, UDP |
| 3 | Network | IP, ICMP, routing |
| 2 | Data Link | Ethernet, MAC addresses |
| 1 | Physical | Cables, signals, hardware |

---

## 2. Key Protocols
| Protocol | Purpose | Port |
|---|---|---|
| HTTP | Web traffic | 80 |
| HTTPS | Secure web | 443 |
| DNS | Domain to IP resolution | 53 |
| DHCP | Assigns IP addresses | 67/68 |
| SSH | Secure remote access | 22 |
| FTP | File transfer | 20/21 |
| SMTP | Sending email | 25 |

---

## 3. Network Security Devices
| Device | Purpose |
|---|---|
| **Firewall** | Filters traffic using rules |
| **IDS** | Detects intrusions, alerts only |
| **IPS** | Detects AND actively blocks intrusions |
| **Proxy** | Intermediary for web requests |
| **VPN** | Encrypted communication tunnel |
| **DMZ** | Buffer zone between internet and internal network |

---

## 4. Common Network Attacks
| Attack | Description |
|---|---|
| **DoS/DDoS** | Overwhelm system with traffic |
| **MITM** | Intercept communication between two parties |
| **Spoofing** | Fake identity — IP, email, DNS |
| **Sniffing** | Capture unencrypted network traffic |
| **Phishing** | Deceptive emails to steal credentials |

---

## 5. Wireless Security
- **WPA2/WPA3** — Current WiFi encryption standards
- **Evil Twin Attack** — Fake access point mimics real one
- **Wardriving** — Scanning for vulnerable WiFi networks
