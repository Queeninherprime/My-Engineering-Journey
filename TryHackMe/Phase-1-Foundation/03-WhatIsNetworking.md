## Key Topics Learned

### What Is A Network?
- A network is when devices are connected
  together to share information
- The internet is one giant network (network of networks)
- Networks can be private or public

### Types of Networks
| Type | Full Name | Description |
|---|---|---|
| LAN | Local Area Network | Small area eg. home, office |
| WAN | Wide Area Network | Large area eg. internet |
| MAN | Metropolitan Area Network | City wide network |

### How Devices Communicate
- Every device needs an IP address
- IP address = device's identity on network
- Devices find each other using IP addresses
- Data travels in packets across networks

### Network Devices
- **Router** — connects networks together
- **Switch** — connects devices in same network
- **Hub** — older, sends to all devices
